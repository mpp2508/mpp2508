# модульное портативное прототипирование (2мм отверстие, 5,08мм шаг)
## modular portable prototyping (2mm hole, 5.08mm pitch)

https://github.com/mpp2508/spcf - как делать модули, все остальное - модули.

Модули крепятся на основу и соединяются dupont перемычками или джамперами, образуя механически монолитный прототип устройства. Он безопасно и удобно транспортируется с места сборки, на место отладки совместно соборудованием, которым он будет управлять. Даже на столе, с таким прототипом удобней работать, чем с клубком из проводов и отладочных плат.
